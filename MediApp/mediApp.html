Aplicación de Recordatorio de Medicamentos (MediApp)

Aplicación Web Progresiva desarrollada para ayudar a los usuarios a gestionar medicamentos y recibir notificaciones de recordatorio.

Problema

Muchas personas olvidan tomar sus medicamentos en los horarios establecidos, lo que puede afectar la efectividad de los tratamientos.

Solución

Se desarrolló una aplicación web progresiva que permite registrar medicamentos, configurar horarios y recibir recordatorios automáticos.

Tecnologías

Flutter
Dart
Laravel
PHP
PostgreSQL

Funcionalidades

  • Registro de usuarios
  • Gestión de medicamentos
  • Configuración de horarios
  • Notificaciones automáticas
  • Historial de medicamentos
